# Bienvenue sur le guide pour PS3Xploit-v3

Pour des guides complets pour les homebrew et Custom Firmware sur d'autres consoles, consultez [hacks.guide](https://hacks.guide).
{: .notice--primary}

Ce guide a pour but de vous expliquer la procédure à suivre afin d'installer des backups PS3 sur votre console.
{: .notice--primary}

Disclaimer : Il est illégal de télécharger et d'installer du contenu que vous ne possédez pas, vous devez posséder les l'oeuvre originale...
{: .notice--warning}

Ce wiki est encore incomplet, l'installation de jeux PSX/PS2/PSP n'étant pas encore documenté, et les outils étant mis à jour fréquemment, il se peut que le wiki devienne rapidement outdated, dans ce cas suivez la procédure avec les versions indiquées.
{: .notice--warning}

Les jeux ne sont pas tous compatibles avec cette méthode. En effet, votre jeux doit:
- Soit se trouver sur le PSN (Exemples : Beyond Two Souls, Doom 3 BFG Edition...)
- Soit, s'ils n'existent qu'en version CD uniquement, avoir une mise à jour de disponible (Exemples : Uncharted 3, Skate 3...) (Un lien sera disponible dans la page dédiée afin de savoir si votre jeu possède une mise à jour)
- Si un jeu ne se trouve dans aucun de ces deux cas (Exemples : Skate 1, Worms Collection...), un Custom Firmware est nécessaire, et si vous vous trouvez sur cette page, cela veut très certainement indiquer qu'il est impossible pour votre console d'en installer un pour le moment sans soudures.

Pour rappel (optionnel) : Si votre console est une PS3 Slim, testez à l'aide de minverchk afin de savoir si votre console est compatible avec les CFW. Si votre console affiche 3.56 ou moins, vous pouvez installer un CFW, mais ce wiki n'est pas dédié à cela. Si c'est une FAT, vous pouvez installer un CFW, si c'est une SuperSlim, il vous faudra obligatoirement suivre ce wiki
{: .notice--primary}

## Important :

Par cette méthode, les jeux s'installeront sur le disque interne de votre console. Dans le meilleur des cas, celui-ci fait originellement 500Go, mais dans le pire des cas, celui-ci fait 12Go. Certains jeux peuvent peser plus de 50Go. Il en faudra le double pour la procédure d'installation (exemple : Uncharted 3 prend 46Go, il vous faudra temporairement 92Go pour l'installer, mais après l'installation, seul 46Go seront utilisés). Dans le cas d'un petit disque dur, il est recommandé de le changer pour un de plus grande taille via cette procédure :_ https://www.playstation.com/fr-fr/get-help/help-library/system---hardware/storage---data/changing-or-upgrading-the-playstation-3-super-slim-hard-disk-dri/. La même procédure est faisable sur PS3 Slim. Il est recommandé de sauvegarder ses données auparavant.
