## Pré-requis matériels : 
- PS3 sous Official Firmware 4.81/4.82 (Paramètres Système -> Informations sur le système)
- Un PC sous Windows (pas testé avec Wine mais il est fort probable que ça soit incompatible)
- Une clé USB / Disque Dur formatée en FAT32 (seul format lisible par la PS3) [GUIFormat x64](http://www.ridgecrop.demon.co.uk/guiformat-x64.Exe)
- Un câble Ethernet et une prise Ethernet sur votre PC
- Un accès à internet pour votre PS3

## Pré-requis logiciels : 

Créez un dossier dédié pour chaque outil
{: .notice--primary}

- PS3Xploit v3 [HAN Tools Support Files](http://www.ps3xploit.com/) (v3 HAN Tools -> Download Support Files)
- [HAN Toolbox](http://www.psx-place.com/resources/han-toolbox.628/)
- [PS3xploit Resigner](https://github.com/PS3Xploit/PS3xploit-resigner/releases)
- [CFW2OFW Helper](https://github.com/friendlyanon/CFW2OFW-Helper/releases)
- [PKGLinker](http://www.psx-place.com/resources/pkg-linker-v1-0-scan-and-serve-packages-to-ps3-from-pc.630/)
- PSNStuff (contient du code propriétaire, un lien ne vous sera pas fourni, mais une recherche Google "PSNStuffX v03.07.00" vous aidera, placez le fichier database (ou une autre database personnalisée) dans le dossier contenant les exécutables)

Il est conseillé d'utiliser un second compte PSN, car il ne fonctionnera plus correctement par la suite.
{: .notice--warning}

## Préparation du périphérique USB :

1. Copiez l'intégralité du contenu de l'archive `.zip` de "HAN Tools Support Files" sur la racine de votre périphérique USB
2. Créez un dossier `packages` sur la racine de votre périphérique USB
3. Copiez le fichier `HAN_Toolbox_vX.X.pkg` dans le dossier `packages` de votre périphérique USB (X.X correspond à la dernière version de HAN Toolbox)

Continuer vers [Installation de HAN](han-install)
{: .notice--primary}
